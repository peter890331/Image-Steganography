# 🕵️ 圖種製作工具 (Image Steganography)

一個輕量級的 Windows 批次檔 (`.bat`) 工具，利用二進位合併技術，將壓縮檔（`.zip`/`.rar`/`.7z`）隱藏在圖片（`.jpg`/`.png`）中。  
製作出來的檔案表面上是一張普通的圖片，但透過特定方式開啟後，可以讀取圖片內部的隱藏壓縮檔。

---

## 🚀 如何製作圖種 (How to Create)

### 1. 準備腳本
下載 Image-Steganography.bat 腳本。

### 2. 執行步驟
雙擊執行腳本，並依照視窗提示操作：
* **步驟 1**: 拖入圖片檔 (`.jpg`/`.png`)，並按下 Enter 確認。
* **步驟 2**: 拖入壓縮檔 (`.zip`/`.rar`/`.7z`)，並按下 Enter 確認。
* **步驟 3**: 輸入欲輸出圖片的相對路徑和檔名 (如 `new.jpg`)，並按下 Enter 確認。

完成！你將獲得一張含有隱藏壓縮檔的圖片。

> ⚠️ **注意：亂碼解決方案**
> 如果下載後執行出現亂碼 (如 ?車鋆...)，請依照以下步驟修正編碼：
> 1. 對 Image-Steganography.bat 按右鍵 -> **編輯**，打開成記事本。
> 2. 在記事本頁面選擇 **檔案** -> **另存新檔**。
> 3. 在另存新檔頁面右下方編碼方式改選為 **ANSI**，覆蓋存檔後即可正常執行。

---

## 🔓 如何解開隱藏壓縮檔 (How to Extract)

由於圖片檔頭 (Header) 的關係，不同的解壓縮軟體開啟方式略有不同：

### 使用 7-Zip (推薦方法)
7-Zip 的圖形介面檢查較嚴格，若直接開啟可能會失敗，請照以下步驟：
1. 將圖片的副檔名改為任意壓縮格式 (如 `new.jpg` -> `new.zip`)。
2. 對圖片按右鍵 -> **7-Zip** -> **開啟壓縮檔** -> **#**。
3. 找到隱藏壓縮檔並按下**解壓縮**，設定路徑提取。

完成！你將獲得圖片內部的隱藏壓縮檔。
