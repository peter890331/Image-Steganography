# 🕵️ 圖種製作工具 (Image Steganography)

一個輕量級的 Windows 批次檔 (.bat) 工具，利用二進位合併技術，將壓縮檔（.zip/.rar/.7z）隱藏在圖片（.jpg/.png）中。    
製作出來的檔案表面上是一張普通的圖片，但透過特定方式開啟後，可以讀取內部的隱藏壓縮檔。

---

## 🚀 如何製作圖種 (How to Create)

1. 下載 Image-Steganography.bat 腳本，並雙擊執行腳本。
2. 依照視窗提示操作：
    * **步驟 1**: 拖入一張圖片 (.jpg/.png)，並按下 Enter 確認。
    * **步驟 2**: 拖入你要隱藏的壓縮檔 (.zip/.rar/.7z)，並按下 Enter 確認。
    * **步驟 3**: 輸入輸出檔名 (如 new.jpg)。
4. 完成！你將獲得一張含有隱藏壓縮檔的圖片。

---

## 🔓 如何解開隱藏壓縮檔 (How to Extract)

由於圖片檔頭 (Header) 的關係，不同的解壓縮軟體開啟方式略有不同：

### 方法一：使用 7-Zip
7-Zip 的圖形介面檢查較嚴格，若直接開啟可能會失敗，請照以下步驟：
1. 將圖片的副檔名改為任意壓縮格式 (如 new.jpg -> new.zip)。
2. 右鍵點選檔案 -> 7-Zip -> 開啟壓縮檔 > -> 選擇 # (解析器模式)**，即可強制忽略圖片標頭並讀取內部的壓縮檔。
